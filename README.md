# Harris Alexandre - Portfólio Landing Page

Landing page e portfólio profissional desenvolvido para destacar serviços de automação, desenvolvimento de sistemas multiplataforma e integrações corporativas.

## 🚀 Tecnologias Utilizadas

- **React.js** (via Vite)
- **Tailwind CSS** (Estilização avançada com Dark Mode e Glassmorphism)
- **React Router** (Navegação SPA)
- **Spline 3D** (Background interativo e animações 3D)

## 📦 Instalação e Execução

Siga os passos abaixo para rodar o projeto localmente:

1. Acesse o diretório do projeto:
   ```bash
   cd harris-alexandre
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. O site estará disponível no seu navegador em `http://localhost:5173`.

## 🛠️ Scripts Disponíveis

- `npm run dev`: Inicia o servidor local com hot-reload.
- `npm run build`: Gera a versão de produção otimizada na pasta `dist/`.
- `npm run preview`: Inicia um servidor local para visualizar a versão de produção (`dist/`).

## 🎨 Funcionalidades em Destaque

- **Parallax 3D Interativo:** Background importado do Spline que reage de forma natural e responsiva ao movimento do cursor do mouse em todas as páginas e seções (através de um controle avançado de `pointer-events`).
- **Glassmorphism UI:** Design focado em transparências (backdrop-blur) para permitir que o 3D seja visualizado enquanto o conteúdo é lido.
- **Responsividade Total:** Interface perfeitamente adaptada do mobile aos maiores monitores.
- **Navegação Fluida:** Rotas gerenciadas pelo React Router para transições imediatas.
